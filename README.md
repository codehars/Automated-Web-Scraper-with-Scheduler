🚀 Automated Web Scraper with Scheduler
🔹 1. Project Overview

You’ll build a system that:

Scrapes data from websites (even dynamic ones)
Runs automatically at scheduled times

Stores data in a database
Exports data to CSV when needed
🔹 2. Tech Stack Explained
Python → Main programming language
BeautifulSoup → For static websites
Selenium → For dynamic websites (JavaScript content)
Celery → For scheduling/background jobs
SQLite → To store scraped data
